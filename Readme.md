# Main Project Backend

This repository contains the backend code for a web application similar to YouTube, with additional features such as tweeting functionality. It is built using Node.js, MongoDB, Express, and other libraries for various functionalities.

## <b>Installation</b>

1. Clone the repository:
   ```bash
   git clone https://github.com/csrahul-112/BackendProject.git
2. Navigate to the project directory:
    <pre>
        cd BackendProject
    </pre>
    or
- navigate to the directory of this project
3. Install dependencies:
    npm install
## <b>Usage</b>
* <b>Development Mode:</b>
    Run the following command to start the server in development mode with hot-reloading:<br> 
    npm run dev

* <b>Production Mode:</b>
    To run the server in production mode, use:<br>
    npm start<br>
    *<b>Right now There is no Script to run the project you can add Yours</b>*
* <b>Testing:</b> Currently, there are no specific tests implemented. You can add your own testing setup as per your project requirements.
    
## Dependencies
* [bcrypt](https://www.npmjs.com/package/bcrypt): Library for hashing passwords securely.
* [cloudinary](https://www.npmjs.com/package/cloudinary): Integration for cloud-based image and video management.
* [cookie-parser](https://www.npmjs.com/package/cookie-parser): Middleware for parsing cookies.
* [cors](https://www.npmjs.com/package/cors): Middleware for handling Cross-Origin Resource Sharing.
* [dotenv](https://www.npmjs.com/package/dotenv): Library for loading environment variables from a .env file.
* [express](https://www.npmjs.com/package/express): Web framework for Node.js.
* [jsonwebtoken](https://www.npmjs.com/package/jsonwebtoken): Library for generating and verifying JSON Web Tokens (JWT).
* [mongodb](https://www.npmjs.com/package/mongodb): MongoDB driver for Node.js.
* [mongoose](https://www.npmjs.com/package/mongoose): Elegant MongoDB object modeling for Node.js.
* [mongoose-aggregate-paginate-v2](https://www.npmjs.com/package/mongoose-aggregate-paginate-v2): Plugin for Mongoose to enable pagination in aggregate queries.
* [multer](https://www.npmjs.com/package/multer): Middleware for handling multipart/form-data, primarily used for file uploads.
